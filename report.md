### Описание проекта:
 Проект представляет собой веб-приложение для управления заметками. Пользователи могут создавать, редактировать и удалять заметки через удобный веб-интерфейс. Приложение поддерживает аутентификацию пользователей и хранит данные в базе данных SQLite. Используется микросервисная архитектура с REST API на [FastAPI](https://fastapi.tiangolo.com/) и шаблонизацией через [Jinja2](https://jinja.palletsprojects.com/). На стороне фронтенда реализован [роутер](frontend/router.py), который обеспечивает навигацию между страницами без перезагрузки с получением необходимых данных через API сервера. Существует пользователь Админ с расширенными правами для управления всеми заметками и пользователями (удалением, изменением и т.д.).
